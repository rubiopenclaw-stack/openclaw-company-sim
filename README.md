# 🦞 OpenClaw 公司模擬器 - AI 員工版 + 遊戲化

一個可以執行真實任務的公司經營模擬遊戲，現在加入遊戲化系統！

## ✨ 新功能 (v3.0 - 遊戲化版)

### 遊戲化系統
- **💰 金幣系統** - 來自 Token 用量轉換和任務獎勵
- **⚡ 能量系統** - 執行任務需要消耗能量，可恢復
- **⭐ 聲望系統** - 完成任務獲得聲望值
- **🏆 成就系統** - 解鎖各種里程碑獲得獎勵

### 抽卡招募系統
- **🎴 普通/稀有/傳說** - 三種稀有度
- **💎 10連抽** - 更優惠的抽卡方式
- **🦞 傳說 AI 員工** - OpenClaw、GPT、MiniMax 等

### 任務關卡
- **📅 每日任務** - 簡單獎勵
- **🏆 週挑戰** - 中等獎勵  
- **⚡ 緊急任務** - 高風險高回報

### 升級系統
- **⬆️ 公司升級** - 累積經驗，解鎖新內容
- **🛠️ 技能升級** - 程式開發、視覺設計、市場營銷等

### 原本功能
- **🦞 OpenClaw AI 員工** - 執行真實任務
- **🤖 自動化任務** - AI 員工自動工作
- **⏱️ 24 小時時間模擬** - 控制時間流逝

## 🚀 快速開始

### 1. 啟動應用

```bash
cd ~/.openclaw/workspace/company-sim
streamlit run app.py
```

### 2. 打開瀏覽器

訪問 http://localhost:8501 (或其他端口)

### 3. 開始遊戲

1. **創建公司** - 輸入公司名稱，選擇類型
2. **🦞 OpenClaw AI 自動加入** - 成為你的 AI 工程師
3. **前往「🎮 遊戲中心」** - 開始抽卡和任務
4. **抽卡招募** - 花費金幣抽取員工
5. **執行任務** - 消耗能量完成任務獲得獎勵
6. **升級公司** - 累積經驗升級公司

## 🎮 遊戲說明

### 💰 Token 轉換
- 每消耗 10 Token = 1 金幣
- Token 用量自動從 API 記錄中讀取
- 可以在側邊欄查看 Token 用量

### 🎴 抽卡系統
| 稀有度 | 機率 | 效率範圍 |
|-------|------|---------|
| 普通 (🟢) | 60% | 0.8-1.0 |
| 稀有 (🔵) | 30% | 1.0-1.3 |
| 傳說 (🟡) | 10% | 1.3-1.8 |

### ⚔️ 任務難度
| 難度 | 能量消耗 | 獎勵倍率 |
|-----|---------|---------|
| 簡單 | 10 | 1.0x |
| 普通 | 20 | 1.5x |
| 困難 | 35 | 2.5x |
| 地獄 | 50 | 4.0x |

- 任務大師 - 完成 100 個任務 (5000💰)

- 小有名氣 - 公司達到 5 級 (2000💰)

## 📁 文件結構

```
company-sim/
├── app.py           # 主應用程序
├── gamification.py  # 遊戲化核心模組
├── requirements.txt # Python 依賴
├── start.sh        # 啟動腳本
├── README.md       # 說明文件
├── game_data.json  # 遊戲數據
├── token_usage.json # Token 用量
├── company.json    # 公司數據
├── employees.json  # 員工數據
├── tasks.json      # 任務數據
├── task_logs.json  # 任務日誌
└── config.json     # 配置文件
```

## 🛠️ 技術栈

- **前端**: Streamlit
- **數據存儲**: JSON 文件
- **AI 任務執行**: subprocess / OpenClaw CLI
- **Token 追蹤**: 從 API 消耗記錄
